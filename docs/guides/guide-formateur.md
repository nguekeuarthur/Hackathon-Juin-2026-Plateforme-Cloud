# Guide formateur

## 1. Se connecter

Allez sur le portail et cliquez **Se connecter avec Office 365** (compte @git.ch).

## 2. Faire une demande groupée pour toute une classe

1. Sur votre tableau de bord formateur, cliquez sur **"Demande groupée"**.
2. Sélectionnez une **classe** ou un **groupe d'étudiants** dans la liste.
3. Choisissez les paramètres communs :
   - Modèle de VM (configuration CPU, RAM)
   - Date de début et date de fin
4. Cliquez sur **"Soumettre la demande groupée"**.

Chaque étudiant recevra une notification et pourra accéder à sa VM une fois la demande validée.

## 3. Suivre les demandes de votre classe

1. Dans votre tableau de bord, accédez à **"Suivi de classe"**.
2. Vous voyez la liste de tous vos étudiants et leurs demandes.
3. Pour chaque étudiant, vous pouvez voir :
   - Le statut de sa demande (en attente, approuvée, refusée)
   - La date de début et de fin de sa VM
   - Le temps restant avant expiration

## 4. Consulter les détails d'une demande étudiante

1. Cliquez sur le nom d'un étudiant dans la liste.
2. Vous accédez au détail de sa demande :
   - Configuration de la VM
   - Historique des actions
   - Consommation approximative

## 5. Aider un étudiant en difficulté

1. Depuis le suivi de classe, sélectionnez l'étudiant.
2. Vous pouvez voir ses informations de connexion SSH si besoin.
3. Si nécessaire, vous pouvez prolonger sa session directement depuis cette interface.

## 6. Voir les statistiques de votre classe

1. Dans votre tableau de bord, allez dans **"Statistiques"**.
2. Vous pouvez consulter :
   - Le nombre total de VM actives dans votre classe
   - La consommation globale estimée

## 7. Gérer les prolongations

1. Allez dans le suivi de classe.
2. Sélectionnez l'étudiant.
3. Cliquez sur **"Prolonger"**.
4. Choisissez la nouvelle durée.
5. Validez.

Une notification sera envoyée à l'étudiant.
