# Guide formateur

## Demande groupée (N machines)
1. Connectez-vous avec votre compte @git.ch
2. Cliquez **Nouvelle demande groupée**
3. Indiquez le template, les dates, et le **nombre de machines**
4. Toutes les machines sont créées identiquement

## Suivi de votre classe
Le dashboard formateur affiche l'état de toutes vos VMs (up/down, ressources).
