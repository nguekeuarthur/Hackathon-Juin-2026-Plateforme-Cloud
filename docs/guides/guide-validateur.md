# Guide validateur

## Approuver ou refuser une demande
1. Connectez-vous (rôle validateur requis)
2. Accédez à **Demandes en attente**
3. Cliquez **Approuver** ou **Refuser** avec un commentaire
4. Le demandeur est notifié automatiquement par e-mail

## Dashboard coûts
Le dashboard affiche les VMs actives, leur consommation et le coût estimé par classe.
